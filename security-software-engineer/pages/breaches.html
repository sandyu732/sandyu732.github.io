Real-world breach timeline

Seven incidents, each teaching a concept you need cold. Read the study page for the concept first, then read the breach to see it fail in production.

2011 · DigiNotar CA compromise breach

A trusted certificate authority was breached and issued fraudulent certificates for google.com, used to intercept Iranian Gmail users' traffic.

Maps to: PKI trust chain

2014 · Heartbleed (OpenSSL) breach

A missing bounds check in OpenSSL's heartbeat extension let attackers read up to 64KB of adjacent memory per request, including private keys.

Maps to: TLS internals & memory safety

2019 · Capital One SSRF breach

A misconfigured WAF let an attacker use SSRF to reach the AWS instance metadata endpoint, stealing IAM credentials and exposing 106M records.

Maps to: SSRF & cloud metadata risk

2020 · SolarWinds / Sunburst breach

Attackers compromised SolarWinds' build pipeline to inject a backdoor into signed software updates, then used forged Kerberos tickets to move laterally.

Maps to: Supply chain & lateral movement

2021 · Log4Shell breach

A single unauthenticated RCE in a near-ubiquitous Java logging library, triggered by a crafted string, put a huge slice of the internet at risk overnight.

Maps to: Dependency risk & patch velocity

2022 · Uber MFA fatigue breach

An attacker spammed push-notification MFA prompts until an employee approved one, then found hardcoded admin credentials on an internal share.

Maps to: MFA weaknesses

2024 · xz-utils backdoor breach

A patient, two-year social-engineering campaign earned commit access to a compression library, hiding a backdoor in the build system targeting sshd.

Maps to: Supply chain & build-system trust
